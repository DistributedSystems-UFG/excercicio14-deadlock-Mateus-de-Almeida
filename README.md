# DeadlockExample
Sem corrigir o código, a saída era:
```
Thread 1: Holding Lock A...   
Thread 2: Holding Lock B...   
Thread 1: Waiting for Lock B...   
Thread 2: Waiting for Lock A...   
```
Acima, observe que as threads foram iniciadas e ambas estão esperando pela (recurso )finalização da outra. Deadlock.  

Corrigindo o código, a saída é: 
```
Thread 1: Holding Lock A...  
Thread 1: Waiting for Lock B...  
Thread 1: Acquired Lock B!  
Thread 2: Holding Lock B...  
Thread 2: Waiting for Lock A...  
Thread 2: Acquired Lock A!  
```
Quebrado o deadlock, a execução de ambas threads é finalizada.
